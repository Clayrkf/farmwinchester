# 🟩 WINCHESTER - Painel de Controle

Sistema de gerenciamento de rotas e farm para o grupo **WINCHESTER** de GTA RP.

## 📋 Funcionalidades

- ✅ Dashboard completo com métricas em tempo real
- ✅ Cadastro de membros (Líder, Gerente, Membro)
- ✅ Registro de rotas com cálculo automático
- ✅ Ranking dos principais farmadores
- ✅ Controle de metas diárias e semanais
- ✅ Histórico completo de registros
- ✅ Configurações personalizáveis
- ✅ Design responsivo (mobile-first)
- ✅ Armazenamento local (LocalStorage)

## 💰 Regras de Divisão

- **Cliente:** 50%
- **Máquina:** 40%
- **Membro:** 5%
- **Família:** 5% + 10% (VIP FAC) = **15%**

## 🚀 Como Usar

1. Acesse o painel pelo link do GitHub Pages
2. Cadastre os membros da organização
3. Registre as rotas realizadas
4. Acompanhe o ranking e metas!

## 🎨 Tema

- **Cor Primária:** Verde Neon (#00FF66)
- **Fundo:** Preto Fosco (#0D0D0D)
- **Estilo:** GTA RP Moderno

## 💾 Dados

Todos os dados são salvos automaticamente no **LocalStorage** do navegador.

## 📱 Responsivo

Funciona perfeitamente em celulares, tablets e desktops.
